📚 README: Phishing & Email Analysis Wrap-up
Role: SOC Analyst / Blue Team
Objective: สรุปเทคนิคการสืบสวนและวิเคราะห์อีเมลฟิชชิ่งเบื้องต้น (Phishing Investigation 101)

1. 📧 Core Email Concepts (โครงสร้างและการส่งอีเมล)
ในฐานะคนเฝ้าระวังระบบ เราต้องรู้ว่าอีเมลทำงานอย่างไรเพื่อจับผิดความผิดปกติ:

Email Address Anatomy: แบ่งเป็น Username (หน้า @) และ Domain (หลัง @) แฮกเกอร์มักจะหลอกตาเราที่ส่วน Domain (เช่น ใช้เทคนิค Typosquatting อย่าง tryhatme.com)

Protocols:

SMTP: หน้าที่ "ส่ง" อีเมล (Send)

POP3: หน้าที่ "ดาวน์โหลด" อีเมลลงเครื่องเดียว (แล้วลบบนเซิร์ฟเวอร์ทิ้ง)

IMAP: หน้าที่ "ซิงค์" อีเมลข้ามหลายๆ อุปกรณ์ (Sync)

DNS: ระบบสมุดหน้าเหลืองที่คอยบอกว่า Mail Server ของผู้รับอยู่ที่ไหน

2. 🕵️‍♂️ Header Investigation (การแกะรอยส่วนหัวอีเมล)
แฮกเกอร์สามารถปลอมแปลงชื่อผู้ส่ง (Spoofing) ได้ แต่หลักฐานทางเทคนิค (Raw Message Source) จะบอกความจริงเสมอ:

X-Originating-IP: คือ IP Address ต้นทางของคอมพิวเตอร์เครื่องแรกที่กดส่งอีเมลนี้มา (ที่อยู่จริงๆ ของแฮกเกอร์)

Authentication-Results: ตราประทับยืนยันตัวตนจากเซิร์ฟเวอร์ ว่าอีเมลนี้ผ่านการตรวจสอบ (เช่น SPF, DKIM) หรือไม่

Defanging (การถอดเขี้ยวเล็บ): กฎเหล็กของ SOC เวลาเขียน Report หรือแชร์ IP/URL อันตราย ต้องเปลี่ยนจุด . เป็น [.] เสมอ เพื่อป้องกันการเผลอกดคลิก (เช่น 192[.]168[.]1[.]1 หรือ hxxp://bad-domain[.]com)

3. 🔬 Payload & Attachment Analysis (การวิเคราะห์เนื้อหาและไฟล์แนบ)
แฮกเกอร์มักจะซ่อน Malware หรือโค้ดอันตรายมาในไฟล์แนบ:

Content-Type & Content-Disposition: ตัวบอกประเภทไฟล์ที่แท้จริงและชื่อไฟล์แนบในโค้ดดิบ

Base64 Encoding: ไฟล์แนบในอีเมลจะถูกเข้ารหัสเป็นตัวอักษรยึกยือ (Base64) หน้าที่ของเราคือใช้ Tools อย่าง CyberChef ในการถอดรหัส (Decode) กลับมาเป็นไฟล์ต้นฉบับ (เช่น PDF หรือ EXE) เพื่อนำไปวิเคราะห์ต่อ

4. 🚨 Types of Malicious Emails (ประเภทของการโจมตี)
Phishing: การหว่านแหส่งอีเมลหลอกลวงทั่วไป

Spear Phishing: การเจาะจงหลอกลวงเป้าหมายแบบเฉพาะเจาะจงบุคคลหรือองค์กร

Whaling: Spear Phishing ขั้นสุดยอดที่เน้นพุ่งเป้าไปที่ผู้บริหารระดับสูง (CEO, CFO)

BEC (Business Email Compromise): การโจมตีขั้นสุดยอดที่แฮกเกอร์ยึดอีเมลของคนในบริษัทได้จริงๆ แล้วสวมรอยใช้อีเมลนั้นไปสั่งโอนเงินหรือขโมยข้อมูลความลับ
